# 📊 Employment Data Statistics Dashboard (Power BI)

---

## 📌 Project Overview
This project presents an *interactive Power BI dashboard* analyzing employment-related data.  
The dashboard provides insights into *income distribution, education levels, occupation types, demographic structure, and settlement patterns*.

It helps stakeholders understand:
- Workforce distribution by gender and education  
- Income variation across education levels  
- Occupational structure of employees  
- Demographic trends such as age and marital status  
- Population distribution by settlement size

# 🎓 Education vs Income Analysis

This section highlights how income varies across different education levels.

| Education Level | Income Contribution |
|----------------|-------------------|
| University | 42.31M |
| Other / Unknown | 33.02M |
| High School | 16.13M |
| Graduate School | 5.28M |

### Insights

* *University-educated individuals earn the highest income*  
* Graduate-level contribution is *surprisingly low*  
* Strong correlation between *education and earning potential*
